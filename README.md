# Description

## Technologies used

- NestJS as the backend framework
- Prisma as the Object-Relational Mapper (ORM)
- PostgreSQL as the database
- Swagger as the API documentation tool
- TypeScript as the programming language

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
